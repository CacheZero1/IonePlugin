## Star Fragments
### Obtaining Star Fragments

Star Fragments can be obtained by Killing monsters in the Overworld.
To get them to drop, the time must be between 13000 and 23000.
They have a [2% = 1/50] Chance to drop.

### Crafting with Star Fragments

Star Fragments can be used to craft the [Cultist Armor Set](#cultist-set).
To craft the whole Cultist Armor Set, 8 Star Fragments are needed.

### Star fragment Usages

When hitting Entities with a Star Fragment, following effects apply:
- Knockback
- Levitation

## Feather Blocks
### Obtaining Feather Blocks

Feather Blocks can be aquired through Crafting.
The Recipe requires the Items to be arranged in the Following Order:

|  |  |  |
| ------ | ------ | ------ |
| Feather (1) | Feather (1) | Feather (1) |
| Feather (1) | Feather (1) | Feather (1) |
| Feather (1) | Feather (1) | Feather (1) |

### Crafting with Feather Blocks

Feather Blocks can be used to craft the [Chicken's Fly Set](#chicken's-fly-set).
To craft the whole Chicken's Fly Armor Set, (24 Feather Blocks / 216 Feathers) are needed.

----

## Ocelot's Blessings
### Obtaining Ocelot's Blessings

Ocelot's Blessings can be obtained, by right clicking (= gifting) Ocelot's with a Gold Nugget.
They have a [100% = 1/1] Chance to drop.

### Crafting with Ocelot's Blessings

Ocelot's Blessings can be used to craft the [Ocelot's Sent Set](#ocelot's-sent-set).
To craft the whole Ocelot's Sent Armor Set, (24 Ocelot's Blessings / 24 Gold Nuggets) are needed.

## Ocelot's Blessing Usages

Right Clicking with an Ocelot's Blessing in the Hand will consume the Item.
Consumption applies following benefits:
- Speed (5 Minutes)
- Jump Boost (5 Minutes)

----

## Galaxy Fragments
### Obtaining Galaxy Fragments

Galaxy Fragments can be obtained by bringing a certain Item to the Cult Lair Dimension.
For Server Owners:
- Cult Lair Dimension (World Name: cultlair)

For Players:
- Cult Lair Dimension (Access: Solve Cult Armor Set Riddle)

### Crafting with Galaxy Fragments

Galaxy Fragments currently don't have a purpose.
They will in the future be able to create the Luna Sword.

### Galaxy Fragment Usages

When hitting Entities with a Galaxy Fragment, following effects apply:
- Knockback

Trying to place down the Galaxy Fragment, will anger (UNKNOWN) and do the following:
- Consume the Item
- Strike the Player

## Installation
### Method 1: Download

Simply go to the **releases** tab and download the latest release

### Method 2: Self Compile

**Since my work is licences under the Creative Commons (By) licence, you are free to modify, sell and redistribute this plugin, as long as Attribution is provided.**

##### Step 1:

Clone this Repo:

```sh
git clone https://github.com/CacheZero1/IonePlugin.git
```

##### Step 2:

Export to JAR
This Project is a standalone Java Project without Gradle or Maven.

----

## Configuration

It is not recommended to edit this Plugin's config.yml.

Important Configuration:
- The Armor Effects work by setting display names, which don't get saved when Players disconnect.
- The cultlair World doesn't have any special environment. For the best experience, build something there!
- This Plugin uses the [ArmorEquipEvent](https://github.com/Arnuh/ArmorEquipEvent) by [Arnuh](https://github.com/Arnuh/). The config.yml file is used by these files.
- The cultlair World can be visited an infinite amount of times.

For the best performance, a database or any other kind of Data saving Method should be used, to make this Plugin work correctly.
